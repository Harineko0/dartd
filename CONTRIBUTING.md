# Contributing

Thanks for your interest in improving `dartd`!

## Getting started
1. Install Dart 3.0+.
2. Fetch dependencies: `dart pub get`.
3. Format and lint: `dart format .` and `dart analyze .`.

## Pull request checklist
- Tests or repro steps for fixes (if applicable).
- No formatting or analyzer errors.
- Keep changes minimal and focused.

## Reporting issues
Open a GitHub issue with:
- What you ran (command + flags).
- Expected vs. actual output.
- Minimal reproduction (project snippet or steps).
